BancoEstado L6

Proyecto desarrollado para el Bootcamp Full Stack Developer SENCE.

Aplicación web para gestionar clientes y cuentas del BancoEstado, 
construida con Node.js y Express.

Funcionalidades
- Listar todos los clientes y sus cuentas
- Filtrar clientes con cuenta RUT
- Agregar nuevos clientes con cuenta RUT o AHORRO
- Eliminar clientes

Cómo correrlo
npm install
npm start

Servidor en http://localhost:3000

Stack
- Node.js
- Express
- Bootstrap 5
